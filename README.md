# Timespent

A personal time tracking app to help you achieve daily time goals for your activities.

## Tech Stack

- Svelte + TypeScript
- Vite
- LocalStorage for data persistence

## Setup
```bash
npm install
npm run dev
```

## Features

- Set time goals with target daily averages
- Log time spent on activities
- Track progress with visual progress bars
- Calculate catch-up schedules when behind on goals
